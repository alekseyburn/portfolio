О себе

Алексей Коршаков

Здравствуйте, я занимаюсь версткой адаптивных/отзывчивых сайтов: лендингов, интернет-магазинов и т.д. Увлекаюсь этим уже 2 года, постоянно развиваясь и совершенствуясь. Имею высшее образование по специализации "Инфокоммуникационные технологии и системы связи". Уровень владения английским языком - чтение технической документации. Успешно закончил 2 курса html&css HTML Academy.

Придерживаюсь в коде чистоты и упорядоченности. Рутинные процессы автоматизирую с помощью сборщика Gulp (префиксы, svg и png спрайты, минификация, извлечение критического CSS и т.д.). Верстаю с соблюдением валидности и доступности. Использую методологию БЭМ для быстрой и легкой разработки. Для ускорения написания стилизации использую препроцессор SCSS. Работаю над общей производительностью веб-сайта. Работа с изображениями в формате webp, ретинизация, адаптивность изображений. Владение js на уровне написания простых скриптов (popup-формы, слайдеры, кастомная валидация форм, работа с объектами), подключение библиотек и их использование, создание простых GSAP анимаций. Также использую git для хранения информации об изменениях в коде.
