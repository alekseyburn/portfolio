О себе

Алексей Коршаков

Здравствуйте, я занимаюсь версткой адаптивных/отзывчивых сайтов: лендингов, интернет-магазинов и т.д. Увлекаюсь этим уже больше года, постоянно развиваясь и совершенствуясь. Имею высшее образование по специализации "Инфокоммуникационные технологии и системы связи". Уровень владения английским языком - чтение технической документации. Успешно закончил 2 курса html&css HTML Academy.

Стараюсь придерживаться в коде чистоты и упорядоченности. Рутинные процессы автоматизирую с помощью сборщика Gulp (префиксы, создание svg-спрайта, минификация, извлечение критического CSS и т.д.). Верстаю с соблюдением валидности и доступности. Использую методологию БЭМ для быстрой и легкой разработки. Для ускорения написания стилизации использую препроцессор SCSS. Кроссбраузерность - IE11, Edge, Firefox, Chrome, Opera, Safari. Оптимизирую прогрессивные веб-метрики (First Contentful Paint (FCP), First Meaningful Paint (FMP), Time to Interactive (TTI)) и работаю над общей производительностью сайта. Работа с изображениями в формате webp (для ie используется полифилл), ретинизация, адаптивность изображений. Владение js на уровне написания простых скриптов (popup-формы, слайдеры, валидации форм), подключение библиотек и их использование, создание простых GSAP анимаций. Также использую git для хранения информации об изменениях в коде и lighthouse для выявления и устранения проблем производительности и доступности.
