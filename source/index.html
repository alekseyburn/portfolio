О себе

Алексей Коршаков

Здравствуйте, я занимаюсь версткой адаптивных/отзывчивых сайтов: лендингов, интернет-магазинов и т.д. Увлекаюсь этим уже больше года, постоянно развиваясь и совершенствуясь. Имею высшее образование по специализации "Инфокоммуникационные технологии и системы связи". Успешно закончил 2 курса html&css HTML Academy.

Стараюсь придерживаться в коде чистоты и упорядоченности. Рутинные процессы автоматизирую с помощью сборщика Gulp (префиксы, создание svg-спрайта, минификация и т.д.). Верстаю с соблюдением валидности и доступности. Использую методологию БЭМ для быстрой и легкой разработки. Для ускорения написания стилизации использую препроцессор SCSS. Кроссбраузерность - IE11, Edge, Firefox, Chrome, Opera, Safari. Работа с изображениями в формате webp (для ie используется полифилл), ретинизация, адаптивность изображений. Владение js на уровне написания простых скриптов(popup-формы, слайдер), подключение библиотек и их использование. Также использую git для хранения информации об изменениях в коде и lighthouse для выявления и устранения проблем производительности и доступности.
